## Español

### Sobre el Proyecto
Esta es una landing page profesional diseñada para **Claudia D'Abundo**, administradora de consorcios en Mar del Plata, Argentina. El sitio está enfocado en transmitir transparencia, profesionalismo y eficiencia, facilitando el contacto directo con propietarios y consorcistas.

### Características Principales
- **Diseño Premium:** Estética moderna y corporativa con enfoque en la experiencia de usuario.
- **Totalmente Responsivo:** Optimizado para dispositivos móviles, tablets y escritorio.
- **Optimización SEO:** Configurado con meta etiquetas para mejorar el posicionamiento en Google.
- **Secciones:** Inicio, Servicios, Sobre Mí, Contacto y acceso a Facturación.
- **Integración con WhatsApp:** Botón flotante para comunicación directa.

### Tecnologías Utilizadas
- **React.js** (Vite)
- **Vanilla CSS** (Diseño a medida)
- **React Icons**

## English

### About the Project
This is a professional landing page designed for **Claudia D'Abundo**, a property manager in Mar del Plata, Argentina. The site focuses on conveying transparency, professionalism, and efficiency, facilitating direct contact with property owners and tenants.

### Key Features
- **Premium Design:** Modern and corporate aesthetics focused on user experience.
- **Fully Responsive:** Optimized for mobile devices, tablets, and desktops.
- **SEO Optimized:** Configured with meta tags to improve Google search rankings.
- **Sections:** Home, Services, About Me, Contact, and Billing access.
- **WhatsApp Integration:** Floating button for direct communication.

### Tech Stack
- **React.js** (Vite)
- **Vanilla CSS** (Custom Design)
- **React Icons**
